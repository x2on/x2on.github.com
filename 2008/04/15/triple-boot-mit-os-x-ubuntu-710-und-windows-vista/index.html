Triple Boot Mit OS X, Ubuntu 7.10 Und Windows Vista

Apr 15th, 2008 | Comments

Dieses Tutorial richtet sich an alle die auf ihrem MacBook Pro neben OS X 10.5 noch Windows Vista SP1 und Ubuntu 7.10 Linux installieren wollen. (sog. Triple Boot)

Grundsätzlich ist dies mittels rEFIT möglich, man muss an manchen stellen ein bisschen tricksen.

1. Voraussetzungen:

  • Ubuntu 7.10 Installations-CD
  • Windows Vista Installations-DVD
  • Bootcamp-Treiber für Windows (Bootcamp Installation ist nicht nötig!)
  • rEFIT
  • Backup gemacht

2. MacOS X Partition verkleinern:

  1. Festplatten-Dienstprogramm starten
  2. Festplatte auswählen
  3. Auf Partitionieren klicken
  4. Mit “+” zwei neue Partitionen anlegen und bei Format “Mac OS X Extended (Journaled)” auswählen
  5. Auf Anwenden klicken
  6. Nun die letzte Partition als “Windows NT Dateisystem” formatieren und als Namen “Windows” eingeben

4. rEFIT installieren:

  1. rEFIT herunterladen: http://refit.sourceforge.net/
  2. Bei der Installation auf Anpassen klicken und die Linux-Treiber mit installieren

5. Ubuntu installieren:

  1. Von der Ubuntu-CD booten
  2. F2 drücken für die Sprachauswahl
  3. Nach kurzer Zeit startet die Ubuntu-Live Version
  4. Auf dem Desktop auf installieren klicken
  5. Setup mit gewünschten Einstellung bis zum Punkt “Partitionieren” durchlaufen
  6. Bei “Partitionieren” Manuell auswählen und auf Weiter klicken
  7. Hier die vorhin erstellte Paritition für Linux auswählen (bei mir /dev/sda3)
  8. auf Edit Parition klicken
  9. Ext 3 auswählen
  10. / auswählen
  11. Auf Ok klicken
  12. Einen Hacken bei “Format” machen
  13. Weiter bis zur Übersicht
  14. Auf Advanced klicken
  15. (hd0) löschen und auf OK Klicken
  16. nochmal auf Advanced klicken
  17. in /dev/sda3 (siehe Punkt 7) ändern
  18. Auf “Installieren” klicken
  19. Nach der Installation neustarten

7. Ubuntu Boot einrichten:

  1. Von der Ubuntu-Live CD booten
  2. Auf System –> Administration –> GNOME Partition Editior klicken
  3. Rechtsklick auf /dev/sd3
  4. Haken bei boot machen
  5. Kann sein das der GNOME Partition Editor dabei abstürzt, sollte trotzdem funktionieren
  6. Falls es danach beim ersten booten hängen bleibt einfach nochmal neustarten

5. Windows installieren:

  1. Vista-DVD einlegen und beim booten auswählen
  2. Bei der Installation die letzte Partition (Windows) formatieren und auswählen
  3. Nach der Installation das SP1 installieren
  4. BootCamp Treiber in Windows installieren
  5. Laufwerk C umbennen in Windows (oder beliebigen Namen)

6. Ubuntu einrichten:

Beim MacBook Pro muss nun der W-LAN Treiber installiert werden, da dieser nicht standardmäßig funktioniert. Für die Grafik sollte der Restricted Treiber “nvidia-glx” aktiviert werden.

  1. Ubuntu updaten:
sudo apt-get update
sudo apt-get upgrade
  1. Ubuntu neustarten
  2. Wlan einrichten:
  3. Terminal öffnen und folgendes eingeben bash wget http://www.x2on.de/dl/mbpinstaller chmod +x mbpinstaller ./mbpinstaller
  4. Optional: Touchpad bei Eingabe temporär ausschalten
1. Öffne ein Terminal

Öffne die xorg.conf mit

`sudo gedit /etc/X11/xorg.conf`

2. In der Synaptics Touchpad Sektion folgende Zeile hinzufügen
3. `Option "SHMConfig" "on"`
4. X mit (Ctrl+Alt+Backspace) neustarten
5. System --> Einstellungen --> Sitzungen öffnen
6. Einen Neuen Eintrag hinzufügen:
7. `syndaemon -t -d`
8. Nun sollte das Touchpad bei einer Tastatureingabe 2 Sekunden das Tippen deaktiviert sein.
  1. Optional: rEFIT Icon ändern: Die Icons beim Start des Mac’s können beliebig verändert werden. Diese sind im Ordner /efi/refit/icons abgelegt.

Viel Spaß mit Ubuntu und Vista auf deinem MacBook Pro.
