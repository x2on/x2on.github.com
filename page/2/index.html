PolarSSL for iOS (iPhone, iPad and iPod Touch)

This is a tutorial for using self-compiled builds of the PolarSSL-library for iOS.

PolarSSL is a light-weight open source cryptographic and SSL/TLS library written in C.

You can build apps with XCode and the official SDK from Apple with this. I also made a small example-app for using the libraries with XCode and the iPhone/iPhone-Simulator. You can also download the precompiled PolarSSL-library (0.14.3).
