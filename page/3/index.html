GoogleSSL Search as Default in Chrome

Since this year you can use Google Search with SSL encryption (https://www.google.com/). With the following URL you can tell Google Chrome to use this encrypted Google Search Site:

https://encrypted.google.com/search?q=%s

If you want use the german Google Site use following URL:

https://encrypted.google.com/search?hl=de&ie;=UTF-8&q;=%s

If you don’t know how to change the search URL: http://www.google.com/support/chrome/bin/answer.py?answer=95653
